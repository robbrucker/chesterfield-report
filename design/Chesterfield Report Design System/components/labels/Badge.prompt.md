Compact uppercase mono label for status and category. Use `breaking` for breaking news, `live` for the pulsing live dot, `civic` (amber) for government/official, `eco` for environment beats.

```jsx
<Badge tone="breaking" dot>Breaking</Badge>
<Badge tone="teal" live>Live</Badge>
<Badge tone="civic">County Board</Badge>
```

Tones: `solid | teal | breaking | civic | eco | neutral`. Pair with `Tag` for selectable topic filters.
